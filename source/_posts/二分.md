# 二分查找

对于一些问题,满足下列条件

1. 由问题抽象出一个区间,使目标值一定在这个区间中
2. 找一个性质,满足:
    1).  区间具有二段性
    2).  答案是二段的分界点

我们可以使用二分来解决.

## 整数二分

首先依据具体问题,找到二段性质,画出二段空间.
![](https://dreamend.oss-cn-shanghai.aliyuncs.com/er_fen1.png)
此时二段分界点即为答案,但此时分界点有两种情况,一是左区间的右顶点,二是右区间的左顶点    
这里是整数二分和实数二分的最大区别.

由此,代码也有所不同
~~~c++
void find()
{
	int l = 0;
	int r = 10000;
	int mid = l + r >> 1;
	if(check()) r = mid;
	else l = mid - 1;
	
}
~~~
